# Appendix: Full Isotopy Cluster Report

All clusters detected at each resource configuration and tier. S = STRONG (t ≥ 10), M = MEDIUM (t ≥ 8), W = WEAK (t ≥ 5).

The clustering algorithm is complete-linkage: a group of *n* lexemes forms a cluster at tier *t* only if every pairwise combination within the group exceeds the tier threshold. Consequently, every detected pairwise isotopic connection is represented in this appendix — either as a standalone two-lexeme cluster, or subsumed into a larger cluster whose membership implies it. A pair appears standalone only when no third lexeme connects to both members above the threshold. The appendix is therefore exhaustive: no detected isotopy is omitted.
